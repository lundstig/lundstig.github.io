Chatter

What is this?

Chatter is a simple chat site where you're connected with another random user.
The conversations are saved so I can find out how people use this, so don't talk about anything sensitive that you wouldn't want me to know :)

How does it work?

The client transmits all of your actions to the server with the help of a some javascript and websockets.
The server is implemented using node.js and WebSocket-Node.
